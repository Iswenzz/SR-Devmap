# Q3

The default weapons on all maps are:
* Rocket (gl_ak47_mp)
* Plasma (gl_g3_mp)

#### ``weapons(<list>)``
Set the Q3 default weapons on spawn seperated by a ";".

Usage Example:
```c
sr\api\_q3::weapons("gl_ak47_mp;gl_g3_mp;");
```
<hr>

#### ``perks(<list>)``
Set the Q3 default perks on spawn seperated by a ";".

Usage Example:
```c
sr\api\_q3::perks("haste;");
```
<hr>

#### ``triggerSection(<id>, <origin>, <width>, <height>, <callback>)``
Create a Q3 trigger section.

**Parameters:**
* id: The section ID.
* origin: The trigger origin.
* width: The trigger width.
* height: The trigger height.
* callback: The callback called on trigger.

Usage Example:
```c
sr\api\_q3::triggerSection("section_1", (0, 0, 0), 150, 100, ::section1);
```
<hr>

#### ``triggerWeapon(<id>, <origin>, <width>, <height>, <weapon>, <?ammo>)``
Create a Q3 weapon trigger.

**Parameters:**
* id: The section ID.
* origin: The trigger origin.
* width: The trigger width.
* height: The trigger height.
* weapon: The Q3 weapon.
* ammo: The weapon ammos, if not specified will give the max ammo.

Usage Example:
```c
sr\api\_q3::triggerWeapon("rocket_1", (0, 0, 0), 150, 100, "gl_ak47_mp", 4);
```
<hr>

#### ``triggerPerk(<id>, <origin>, <width>, <height>, <perk>, <?time>)``
Set the Q3 default perks on spawn seperated by a ";".

**Parameters:**
* id: The section ID.
* origin: The trigger origin.
* width: The trigger width.
* height: The trigger height.
* perk: The perk.
* time: The time the perk last, if not specified it will last forever.

Usage Example:
```c
sr\api\_q3::triggerPerk("haste_1", (0, 0, 0), 150, 100, "haste", 4);
```
<hr>

#### ``switchToQ3Weapon(<name>)``
Switch to a Q3 weapon.

Usage Example:
```c
player sr\api\_q3::switchToQ3Weapon("gl_ak47_mp");
```
<hr>

#### ``takeQ3Weapon(<name>)``
Take off a Q3 weapon.

Usage Example:
```c
player sr\api\_q3::takeQ3Weapon("gl_ak47_mp");
```
<hr>

#### ``takeAllPerks()``
Take off all perks.

Usage Example:
```c
player sr\api\_q3::takeAllPerks();
```
<hr>

#### ``takeQ3Perk(<id>)``
Take off a Q3 perk.

Usage Example:
```c
player sr\api\_q3::takeQ3Perk("haste");
```
<hr>

#### ``giveQ3Weapon(<name>, <?ammo>)``
Give a Q3 weapon with specific ammo.
If the ammo parameter is not specified it will give the max ammo.

Usage Example:
```c
player sr\api\_q3::giveQ3Weapon("gl_ak47_mp", 4);
```
<hr>

#### ``giveQ3Perk(<id>, <?time>)``
Give a Q3 perk.
If the time parameter is not specified the perk will last forever.

Usage Example:
```c
player sr\api\_q3::giveQ3Perk("haste", 4);
```
<hr>
