# SR Devmap

![](https://i.imgur.com/1fWSIdh.png)

A lightweight library included in all SR mod repositories and compiled into mod.ff, enabling solo offline play via CoD4's devmap mode. Load and run maps from any major mod — Speedrun, Deathrun, BattleRoyale, Surf, CJ, and more.

SR-Devmap is useful for both players who want to run maps offline and mappers who want to test their work or experiment with the SR API during development, all from within a minimal local environment.
